## Development Setup
Clone the repository and run the following commands:

```
git clone https://github.com/colonal/checkout_payment.git
cd checkout_payment
```

Create .env file in root project ./.env
add values
```
SECRET_KEY=sk_test_...
PUBLISHABLE_KEY=pk_test_...
CLIENT_Id=AdCe1dq...
CLIENT_SECRET=EJ537kLmUsZkIe6dk_...
```
run command
```
flutter pub get
flutter run 
```
